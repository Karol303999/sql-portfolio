# sql-portfolio/EXCEL

# Projekt SQL

Projekt SQL do analizy danych pracowników obejmujący zapytania z agregacjami,
filtrowaniem, JOINami, podzapytaniami oraz CTE.

# Projekt Excel – analiza sprzedaży w EURO z wykorzystaniem tabel przestawnych i filtrów.

Projekt polega na analizie danych sprzedażowych w Excelu z wykorzystaniem tabeli
przestawnej prezentującej sprzedaż produktów w EURO z możliwością filtrowania
po opiekunach

# Projekt Excel obejmujący analizę danych pracowników, normalizację struktury danych, tabele przestawne, obliczenia finansowe oraz wizualizację wyników.

Projekt polega na analizie danych pracowników na podstawie arkusza Excel.
Zakres prac obejmuje podział danych na logiczne struktury (relacje),
utworzenie tabeli danych, analizę wynagrodzeń z wykorzystaniem tabel
przestawnych, obliczenia dochodu na członka rodziny, wyliczenie wieku
pracowników oraz prezentację wyników na wykresie.
